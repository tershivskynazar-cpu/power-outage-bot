### 🛠️ Інструкція розгортання:

#### 1. GitHub (2 хвилини)
```bash
git init
git add .
git commit -m "Power Outage Bot - Ready for deploy"
git remote add origin https://github.com/YOUR_USERNAME/power-outage-bot.git
git push -u origin main
```

#### 2. Render (3 хвилини)
1. **render.com** → "New Web Service"
2. **Connect GitHub** → виберіть репозиторій
3. **Налаштуйте**:
   - Runtime: Python 3
   - Build: `pip install -r requirements.txt`
   - Start: `python bot.py`
